# Hyprland Dotfiles

A refined configuration for a robust Arch Linux setup, tailored for performance and productivity. This repository encapsulates the core elements of my Hyprland desktop, offering a clean, efficient, and cohesive environment.

---

### Key Components

My configuration encompasses a carefully curated selection of applications and system utilities, ensuring a seamless and responsive user experience.

```
.
├── Core Utilities
│   ├── alacritty          - Fast, GPU-accelerated terminal emulator
│   ├── dunst              - Lightweight and customizable notification daemon
│   ├── fastfetch          - NeoFetch-like system information tool
│   ├── gammastep          - Screen color temperature adjustment (f.lux alternative)
│   ├── htop               - Interactive process viewer
│   ├── ibus               - Intelligent Input Bus (for input methods)
│   ├── menus              - Custom application launchers/scripts
│   ├── nvim               - Modern, extensible Vim-based text editor
│   ├── brightnessctl      - Command-line utility for screen brightness control
│   ├── rofi               - Versatile application launcher and dmenu replacement
│   ├── waybar             - Highly customizable Wayland bar for status display
│   └── firefox            - Privacy-focused web browser
│
├── Applications & Tools
│   ├── calcurse           - Command-line calendar and organizer
│   ├── gimp               - GNU Image Manipulation Program
│   ├── godot              - Open-source game engine
│   ├── JetBrains          - Integrated Development Environments (e.g., IntelliJ, PyCharm)
│   ├── obsidian           - Knowledge base and markdown editor
│   ├── nwg-look           - GTK theme and icon settings utility for Wayland
│   ├── peaclock           - Simple clock utility
│   ├── spicetify          - Spotify client customizer
│   └── flatpak            - Universal software packaging system
│
├── Database Management
│   ├── mariadb            - Robust relational database server
│   └── mysql-workbench    - Visual database design and management tool
│
├── System Drivers & Hardware
│   ├── envycontrol        - NVIDIA Optimus laptop GPU switcher

│   ├── nvidia-settings    - NVIDIA X Server Settings utility (570.153.02-1)

│   └── lm-sensors         - Hardware monitoring utilities
│
├── Hyprland Ecosystem
│   ├── hyprlock           - Wayland screen locker for Hyprland

├── System Tray Integrations

├── Screenshot Tools

│
├── File Management
│   └── nemo               - Powerful and feature-rich file manager
│
├── Clipboard Management
│   ├── cliphist           - Wayland clipboard history utility
│   └── wl-clipboard       - Command-line utilities for Wayland clipboard

└── Audio Control
    ├── PipeWire           - Modern multimedia framework
    ├── WirePlumber        - Session and policy manager for PipeWire
    ├── pavucontrol        - PulseAudio Volume Control (for PipeWire)
    └── playerctl          - Command-line utility for controlling media players
```
